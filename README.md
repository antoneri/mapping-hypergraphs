# Mapping flows on hypergraphs

## Requirements

Requirements are specified in the included `Pipfile`:

- Python 3.8
- Infomap Python package 1.2.0

Install with `pipenv`:

```bash
pipenv install
```

## Running

Main entrypoint is in `hypergraph/main.py`.

Activate the environment with
```bash
pipenv shell
```

Then, run the code
```bash
python -m hypergraph --help
```
